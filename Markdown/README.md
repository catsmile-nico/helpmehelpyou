## How to hide/show content on click

### __RAW__

```md
<details>
    <summary>Click to show details...</summary>
    Some content
    - listitem
    - listitem
</details>
```

### __PREVIEW__
<details>
<summary><span style="color:lightblue">Click to show details...</span></summary>

Some content  
- listitem
- listitem
</details>
